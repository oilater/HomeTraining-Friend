### 개발 목적

평상시 목표 개수를 정해 놓고 홈트를 하는 편입니다.
운동 별로 목표 개수를 설정하고, 틈틈히 기록할 수 있다면 좋지 않을까란 생각에 개발을 시작하게 되었습니다.


### 개발 기간
 - 3월 11월(월) - 15일(금) 약 5일


### 개발 언어

- Swift로 개발하였습니다.
- 서버는 없으며, SwiftData를 사용하여 데이터를 저장하였습니다.
